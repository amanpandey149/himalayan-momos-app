# 🍜 Himalayan Darjeeling Momos - Food Ordering App

A modern food ordering web application built with React.js. Users can browse menu items, add to cart, and securely place orders using QR-based payment and Razorpay integration.

---

## 📌 About
This project is a complete restaurant ordering system that allows users to select food items, manage their cart, and make payments through UPI QR or Razorpay before confirming their order.

---

## ✨ Features
- 🛒 Add to Cart system
- 💰 Dynamic total price calculation
- 📱 QR Code payment with auto-filled amount
- 💳 Razorpay payment integration
- ✅ Payment verification before order confirmation
- 📦 Order summary and checkout flow
- 📍 Address & contact form
- 📱 Responsive and clean UI

---

## 🛠️ Tech Stack
- React.js (Vite)
- JavaScript
- Tailwind CSS
- Razorpay Payment Gateway
- UPI Payment Integration
